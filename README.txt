Le projet s'articule autour du index.php
- Une fonction de connexion au compte administrateur est en place.
- Dans le cas où il n'y a pas de compte admin, la fonction créer un nouveau compte prend le relai.
- De l'index, on peut (théoriquement) ajouter un employé à la base de données.
- Une fois ajouté, on peut lister les employés saisis au préalable.
- La page index.php est responsive.
